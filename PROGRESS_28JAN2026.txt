                              PROJECT OVERVIEW

🎯 Objective:
-------------
1. To develop a Thai-language voice assistant system capable of real-time interaction.
2. To integrate the assistant with a mobile robot to perform fundamental actions 
   through voice commands.
3. To extend the system with a camera module for object detection and reporting.

📋 Scope (Fixed, Post-Defense):
-------------------------------
- Experiments in a controlled room environment.
- Support voice commands for robot movement + simple object-finding tasks.
- Provide responses via speech and text.
- Evaluation metrics: Accuracy, Latency, Task Success Rate.

Quick Start (Tailscale Mode - Recommended):
-------------------------------------------

1. SSH เข้า server:
   ssh user@<server-ip>

2. เปิด tmux session:
   tmux new -s vora

3. Start VORA:
   cd /home/user/vora_project/VORA/VORA
   source vora_env/bin/activate
   export VORA_API_PORT=8080 VORA_HTTPS=true
   uvicorn app.main:app --host 127.0.0.1 --port 8080 &

4. Start Tailscale Serve:
   tailscale serve --bg 8080

5. Access from any device (with Tailscale):
   https://user.tail87d9fe.ts.net/app
