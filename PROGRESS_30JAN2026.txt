**Diagnosis:**
```bash
# On MyAGV:
ros2 topic echo /cmd_vel
# Shows messages:
linear:
  x: 0.1  # Forward velocity
  y: 0.0
  z: 0.0
angular:
  x: 0.0
  y: 0.0
  z: 0.0

# But robot doesn't move!
ros2 topic info /cmd_vel
# Subscribers: 0  ← Problem!
```

**Root Cause:**
- myagv_odometry package exists but no launch file
- ros2 launch myagv_odometry myagv.launch.py → File not found
- Need to find or create motor controller node
